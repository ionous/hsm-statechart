hsm_context HsmContextAlloc ( size_t  size)

Helper to create a new context object.

It will be auto-freed when popped.

Parameters:
sizeTotal size of the user context structure.
Returns:
Newly allocated context data, NULL if the allocation failed.

Sets the hsm_context_rec::popped handler to an internal free function. You can also allocate your objects on your own if you want more control.
